# Bank Marketing ML Analysis

A machine learning project to predict whether a client will subscribe to a term deposit using real-world bank marketing data.

---

## 📊 Project Overview
This project analyzes customer data and applies multiple machine learning models to predict subscription outcomes, helping improve marketing strategies.

---

## ⚙️ Technologies Used
- Python
- Pandas & NumPy
- Scikit-learn
- Matplotlib / Seaborn

---

## 🔍 Workflow
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training (Logistic Regression, Decision Tree, etc.)
- Model Evaluation

---

## 📈 Results
- Best Model: LIGHTGBM 
- Accuracy: 90%
- Precision: 74%
- Recall: 74%

---

## 📁 Dataset
The dataset used in this project is included in the repository (`bank-full.csv`).
